# STRAT-AI – Race Strategy Assistant

## 🚀 Problem

Race teams must make fast decisions about pit stops under uncertainty. Choosing the wrong moment can cost positions or the race.

## 🧠 Solution

STRAT-AI is an AI-powered race strategy assistant that simulates future race scenarios and recommends whether to pit or stay out.

## ⚙️ How It Works

* Takes inputs like tire age and pit stop time loss
* Simulates future lap performance
* Compares strategies (pit vs stay)
* Outputs:

  * Decision
  * Explanation
  * Confidence score

## 🤖 AI Approach

This project uses simulation-based decision-making and is designed to integrate with IBM Granite models for generating explainable strategy insights.

## 🏎️ Why It Matters

Race strategy decisions are critical and time-sensitive. STRAT-AI helps improve decision-making by providing clear, data-driven recommendations.
